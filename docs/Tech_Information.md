
# Computers

## Electronic Computer
Pros:

* cheap
* medium fragile

Cons:

* least efficient computer (smallest output per energy)
* lowest output per cell


## Crystal Computer
Pros:

* most efficient computer (highest output per energy)
* quite cheap
* least fragile

Cons:

* Requires rare materials


## Quantum Computer
Pros:

* highest output per cell 

Cons:

* expensive
* high energy consumption
* most fragile
